# Tabs

## Project Title

**AI-Powered Customer Support & Service Automation using Salesforce Service Cloud, Flow, Prompt Builder, and Agentforce**

---

# Overview

Tabs in Salesforce provide users with quick access to objects, records, reports, and dashboards. For this project, standard and custom tabs were configured to enable customer support representatives to efficiently manage customers, products, orders, support cases, and warranty claims from a centralized application.

---

# Standard Tabs

The following standard Salesforce tabs are used in the project:

| Tab        | Description                                                                          |
| ---------- | ------------------------------------------------------------------------------------ |
| Home       | Displays the Salesforce home page with recent items and notifications.               |
| Accounts   | Stores customer account information.                                                 |
| Contacts   | Maintains customer contact details.                                                  |
| Cases      | Allows support agents to create, assign, update, and resolve customer support cases. |
| Products   | Stores product information used during customer support.                             |
| Reports    | Provides analytical reports on customer support activities.                          |
| Dashboards | Displays key performance indicators (KPIs) and support metrics.                      |

---

# Custom Tabs

The following custom tabs were created to support business-specific requirements:

| Tab             | Purpose                                                |
| --------------- | ------------------------------------------------------ |
| Customer Orders | Stores customer order details and order history.       |
| Warranty Claims | Manages product warranty requests and approval status. |

---

# Tab Configuration

The tabs were configured using Salesforce App Manager and Lightning App Builder with the following settings:

* Added relevant tabs to the Customer Support application.
* Configured tab visibility based on user profiles.
* Organized tabs for easy navigation.
* Assigned appropriate permissions for different user roles.

---

# Benefits of Using Tabs

The configured tabs provide several advantages:

* Quick access to customer and support records.
* Improved navigation within the Salesforce application.
* Better organization of business data.
* Reduced time required to locate information.
* Enhanced productivity for customer support agents.

---

# Expected Navigation Flow

A support agent can typically navigate through the application as follows:

1. Open the **Home** tab.
2. Search for the customer using the **Accounts** or **Contacts** tab.
3. Review the customer's **Orders** and **Warranty Claims**.
4. Create or update a **Support Case**.
5. Use AI-generated summaries through Prompt Builder.
6. Monitor performance using **Reports** and **Dashboards**.

---

# Conclusion

Salesforce tabs provide a structured and user-friendly navigation experience for the AI-Powered Customer Support & Service Automation system. By organizing standard and custom objects into dedicated tabs, the application enables support agents to efficiently access customer information, manage support cases, and utilize AI-powered features within a unified Salesforce environment.
