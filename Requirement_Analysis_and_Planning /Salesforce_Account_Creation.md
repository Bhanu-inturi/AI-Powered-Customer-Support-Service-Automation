# Salesforce Account Creation

## Project Title

**AI-Powered Customer Support & Service Automation using Salesforce Agentforce**

## Purpose

A Salesforce Developer Edition account was created to provide a cloud-based environment for developing, testing, and demonstrating the AI-Powered Customer Support & Service Automation project. The Developer Edition includes the tools and features required to configure Salesforce applications and AI-powered functionality.

---

## Account Creation Steps

### Step 1: Register for Salesforce Developer Edition

1. Visit the Salesforce Developer sign-up page.
2. Fill in the required details:

   * First Name
   * Last Name
   * Email Address
   * Role (Developer)
   * Company/College Name
   * Country
   * Postal Code
   * Username (must be unique and in email format)
3. Submit the registration form.

---

### Step 2: Verify the Email Address

1. Open the verification email sent by Salesforce.
2. Click **Verify Account**.
3. Create a secure password.
4. Select a security question and provide an answer.
5. Complete the account activation process.

---

### Step 3: Access the Salesforce Environment

After successful verification, log in to the Salesforce Developer Edition account. The Salesforce Setup page serves as the primary workspace for configuring objects, automation, security, and AI features.

---

## Initial Configuration

After account creation, the following setup tasks were completed:

* Accessed Salesforce Setup.
* Enabled Service Cloud features.
* Created a dedicated application for the project.
* Configured navigation and page layouts.
* Verified access to Flow Builder, Prompt Builder, and Agentforce.
* Prepared the environment for project development.

---

## Features Available in the Developer Edition

The Developer Edition account provides access to:

* Salesforce Service Cloud
* Lightning Experience
* Object Manager
* Flow Builder
* Lightning App Builder
* Prompt Builder
* Agentforce
* Validation Rules
* Profiles and Permission Sets
* Reports and Dashboards

---

## Outcome

The Salesforce Developer Edition account was successfully created and configured. It serves as the development environment for implementing the AI-Powered Customer Support & Service Automation system, enabling the creation of custom objects, automation workflows, AI-powered prompts, and secure user access.

---

## Conclusion

Creating a Salesforce Developer Edition account is the first step in building Salesforce applications. It provides a secure, cloud-based platform for development, testing, and demonstration while offering access to Salesforce Service Cloud and AI capabilities required for this project.
